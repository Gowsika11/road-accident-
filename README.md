# Road Accident Analysis

## Overview
This project analyzes road accident data to identify patterns, trends, and factors contributing to accidents. It uses Machine Learning techniques to predict accident outcomes and provides an interactive interface using Streamlit.

## Features
- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Machine Learning model for prediction
- Interactive Streamlit web application
- Model saved using Pickle (.pkl)

## Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Streamlit
- Pickle

## Project Structure

```
road-accident-analysis/
│── 9may.ipynb              # Jupyter Notebook
│── model.pkl               # Trained Machine Learning model
│── streamlit_app.py        # Streamlit application
│── requirements.txt        # Required Python packages
│── README.md               # Project documentation
```

## Installation

1. Clone the repository

```bash
git clone https://github.com/Gowsika11/road-accident-.git
```

2. Navigate to the project folder

```bash
cd road-accident-
```

3. Install the required packages

```bash
pip install -r requirements.txt
```

## Run the Application

```bash
streamlit run streamlit_app.py
```

## Machine Learning Workflow

The trained model predicts road accident outcomes based on the provided input features. The Streamlit application allows users to interact with the model through a simple web interface.
